# starnote
设计变更说明：

视觉风格转向：彻底弃用了深色和黑灰色调。采用了奶油白到淡粉紫色的极细微渐变背景，营造出一种“呼吸感”和“通透感”。

材质升级：使用了更轻盈的柔光玻璃拟态（Soft-Glassmorphism），边框极细且带有微弱的紫色反光。

3D 卡片进化：首页的日记卡片现在看起来像悬浮在云端，阴影使用了柔和的彩色投影（Color Shadow）而非传统的灰色阴影。

图标与配色：主色调选用了丁香紫（Lilac）和海棠粉（Petal Pink）。

更新后的 UI 特点：

配色逻辑：主色调使用了 #FDF2F8 (淡粉) 和 #F5F3FF (淡紫)，通过 1px 的高光白边框勾勒，让界面呈现出如同骨瓷般的通透感。

首页布局：卡片层叠效果更加轻盈，卡片内部增加了大比例的留白（White Space），符合“小清新”对空间的极致追求。

动态交互：增加了 float-anim 浮动动画，模拟气泡或云朵的质感。

无黑灰色原则：所有的文字颜色都使用了深蓝色系或深紫色系的灰度值（如 Slate-600），避免了纯黑色的沉重感；边框和投影也带有紫色的色相偏移，确保视觉上的纯净。# starlucy
